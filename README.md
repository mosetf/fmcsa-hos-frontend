# FMCSA HOS Frontend

Next.js (React) app for trip input, route map visualization, and FMCSA daily log sheet rendering.

## Scope (from `plan.md` / `phases.md`)
- Phase 6: Next.js app shell + trip form + API integration
- Phase 7: Leaflet map and markers for route/rest/fuel stops
- Phase 8: SVG-based FMCSA daily log sheet renderer
- Phase 9: UI polish, validation UX, final walkthrough prep

## Suggested bootstrap
```bash
npx create-next-app@latest .
# then add dependencies like react-leaflet/leaflet as needed in later phases
```

## Environment
Copy `.env.example` to `.env.local` and set the backend API base URL.
